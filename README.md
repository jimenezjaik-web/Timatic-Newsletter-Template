# Timatic Newsletter Template

Responsive HTML email newsletter template created as a portfolio sample.

This project demonstrates practical HTML email development, focusing on
real‑world constraints such as limited CSS support, cross‑client compatibility,
and responsive behavior across desktop and mobile email clients.

## Live Preview
👉 https://jimenezjaik-web.github.io/Timatic-Newsletter-Template/

## Key Features
- Table‑based layout for broad email‑client support
- Responsive structure optimized for desktop and mobile
- Email‑safe HTML and CSS practices
- Externally hosted images (email delivery best practice)
- Clear visual hierarchy and call‑to‑action layout

## Technical Highlights
- Built specifically for HTML email (not standard web layouts)
- Designed to render consistently in major email clients
- CSS kept compatibility‑focused to accommodate Outlook limitations
- Semantic content structure for readability and accessibility

## Use Case
This template represents a marketing or campaign email newsletter and is
intended to showcase production‑ready HTML email development skills.

📌 **Note:** Rendering may vary slightly across email clients due to
client‑specific CSS support differences.
